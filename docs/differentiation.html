18.1 Basic Derivative Practice

  1. \(W(r) = r^3 + 5r - 12\)
  2. \(f(x) = x^2 - 3\ln x\)
  3. \(P(t) = 4t^2 + 7\sin t\)
  4. \(f(x) = 1/x^2 + 5\sqrt{x} - 7\)
  5. \(s(x) = 2e^{x} + x^2\)
  6. \(h(\theta) = 1/\sqrt[3]{\theta}\)
  7. \(b(t) = t^2 + 5\cos t\)

18.2 More Basic Derivative Practice

  1. \(g(x) = -\frac{1}{2}\left(x^5 +2x -9 \right)\)
  2. \(s(t) = 6t^{-2} + 3t^3 - 4t^{1/2}\)
  3. \(q(x) = 3x - 2\cdot 4^x\)
  4. \(y(x) = \sqrt{x}(x+1)\)
  5. \(P(t) = 3000(1.02)^t\)
  6. \(f(x) = 2^x + x^2 + 1\)
  7. \(d(r) = Ae^{r} - Br^2 + C\)
  8. \(s(t) = t^2 + 2\ln t\)
  9. \(g(x) = 2x - \frac{1}{\sqrt[3]{x}} + 3^x - e\)
  10. \(y(g) = 5\sin g - 5g + 4\)

18.3 Product Rule and Quotient Rule Practice

  1. \(s(t)=(t^2+4)(5t-1)\)
  2. \(y(x)=x^2\ln x\)
  3. \(g(x)= \frac{25x^2}{e^x}\)
  4. \(h(t)=\frac{t+4}{t-4}\)
  5. \(y(x)=x^2\cos x\)

18.4 Chain Rule Practice

  1. \(g(z)=(z^2+5)^3\)
  2. \(g(m) = 200 e^{0.12m}\)
  3. \(h(x) = e^{5x^2+1}\)
  4. \(g(x) = \sqrt{1+x^3}\)
  5. \(f(t) = \cos(2t^3+5t^2)\)
  6. \(k(x) = \ln(4x^2)\)

18.5 More Chain Rule Practice

  1. \(f(x)=\sin(2x)\)
  2. \(P(t)=e^{2t}\)
  3. \(y(x)=x e^{3x}\)
  4. \(f(z)=\ln(z^2+1)\)
  5. \(R(t)=(\sin t)^5\)
  6. \(f(t)=\sin \sqrt{e^{t}+1}\)
  7. \(g(y)=e^{2e^{y^3}}\)
